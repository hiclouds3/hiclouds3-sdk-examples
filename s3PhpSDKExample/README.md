# hicloud S3 PHP SDK Examples

## Prerequisites
- PHP >= 8.1
- Composer

## Getting Started

### Setup

* Fill in AccessKey, SecretKey in「sample/client.php」

* Fill in user information in「sample/config.php」


### Building the project

```sh
$ cd sample/

$ composer install
```

### Running
- S3Sample: Demo of simple S3 operations

    ```sh
    $ cd sample/
    $ php S3Sample.php
    ```

- SerialTest

    ```sh
    $ cd sample/
    $ php PHPSerialTest.php
    ```
