# hicloud S3 Ruby SDK

## Prerequisites

- Ruby
- Install [AWS SDK for Ruby - Version 3](https://github.com/aws/aws-sdk-ruby)

  - ```gem install aws-sdk --version '~> 3'```

## Getting Started

### Setup

* Fill in AccessKey, SecretKey, User information in「config.json」

### Running
- SerialTest

    ```sh
    $ cd sample/
    $ ruby ./s3_serial_testing.rb
    ```
