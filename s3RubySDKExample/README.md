# hicloud S3 Ruby SDK Examples

## Prerequisites

- Ruby

## Getting Started

### Setup

* Fill in AccessKey, SecretKey, User information in「config.json」

### Running
- Install dependencies
  ```
  $ gem install bundler:2.4.22

  $ cd sample/
  $ bundle install
  ```

- SerialTest

    ```sh
    $ bundle exec ruby ./s3_serial_testing.rb
    ```
